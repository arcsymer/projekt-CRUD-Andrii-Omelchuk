FoodReview – Recenzje dań restauracji
Projekt FoodReview to prosta aplikacja webowa umożliwiająca dodawanie, edytowanie, usuwanie oraz przeglądanie recenzji potraw w restauracjach.
Aplikacja działa w architekturze frontend + backend z pełnym systemem CRUD oraz autoryzacją użytkowników.

Struktura projektu
foodreview/
├── index.html
├── loginreg.html
├── reviewadd.html
├── tops.html
├── profile.html
├── style.css
├── script.js
├── db.php
├── api.php
├── particles.js
└── README.md

Technologie
Frontend: HTML5, CSS3, JavaScript (czysty JS, dynamiczne ładowanie danych, animacje)
Backend: PHP 8+ (REST API)
Baza danych: MySQL / MariaDB

Funkcjonalności
Rejestracja i logowanie użytkowników
Dodawanie nowych recenzji z:
	nazwą dania
	nazwą restauracji
	adresem restauracji
	datą wizyty
	oceną (1–10)
	rekomendacją
	komentarzem (opcjonalnym)
Edycja i usuwanie własnych recenzji
Wyświetlanie ostatnich recenzji
Top-listy:
	5 najczęściej odwiedzane restauracje
	5 najwyżej ocenione dania
Dynamiczne wyświetlanie ocen
Animowane tło (particles.js) i gradient
Responsywny, nowoczesny design
Ukryty pasek przewijania w sekcji recenzji
